native-qemu appliance

This is the host documentation service. It is intentionally available only on the guest-facing bridge.

Configuration and guest-service guide

Host administration

Use SSH to reach the host and inspect /var/log/native-qemu.log, edit /etc/native-qemu/config.toml, then run lbu commit to persist changes.

Shared folder

When [shared_folder] is enabled, mount it in a Linux guest with:

mount -t virtiofs native-qemu-share /mnt/shared

Windows guests need the virtio-fs driver. An enabled SMB share is an alternative for guests without that driver.

Safety

The guest disk image is not mounted by the host while QEMU is running. Move files through the shared folder or configured SMB shares instead.
